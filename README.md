# Chat Server-Client in Java

A chat server and client in Java.

Can do;
- Choose name and server to login
- Create chat groups
- Chat private
- Send message to all
- Chat in group

Can't do;
- No authentication or encryption (database or web service can be added for authentication)
- No accounting
